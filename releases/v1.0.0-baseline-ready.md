# Aurora v1.0 Polaris — v1.0.0 baseline-ready

This release marks the first frozen baseline slice of Aurora v1.0 Polaris.

## Included in this release

- `polaris-taxonomy-pack v1.0.0 baseline-ready`
- `polaris-mvp-module-registry-pack v1.0.0 baseline-ready`
- `polaris-mvp-policy-decision-pack v1.0.0 baseline-ready`
- `polaris-mvp-event-pack v1.0.0 baseline-ready`
- `polaris-baseline-bundle v1.0.0 baseline-ready`

## Baseline contents

- controlled taxonomies
- 12 MVP module manifests
- 6 policy-decision seed instances
- 7 MVP event seed instances
- dependency mapping
- cross-pack validation reporting
- artifact manifest for the frozen baseline bundle

## Design intent

This repository is published as a governance-first reference bundle for a regulated multi-agent MVP. It is intended as a specification and baseline artifact set, not as a production runtime.

## Deliberately open areas

The following remain intentionally open in this release:
- concrete policy-pack contents
- numerical risk-scale definition
- integrity-marker method
- operational connector approvals
- OpenAPI/gRPC derivations

## Release status

`v1.0.0 baseline-ready`
