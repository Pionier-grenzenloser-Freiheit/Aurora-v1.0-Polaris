# Jan-MiroFish-Companion-Repo

Dieses Repository ist ein deutsches Jan-MiroFish-Companion-Repo: ein sicherer, dokumentierter Arbeitsraum für die Nutzung, Prüfung und Einordnung von MiroFish-Ergebnissen durch Jan, Aurora und Codex.

## Wichtige Einordnung

Dieses Repository enthält aktuell **nicht** den vollständigen MiroFish-Quellcode. Es ist keine ausführbare MiroFish-App, kein Fork des Upstream-Projekts und keine Installationsanleitung für eine produktive MiroFish-Instanz.

Das echte MiroFish-Upstream-Projekt muss als externe Quelle separat geprüft werden, bevor technische Aussagen zu Installation, Startbefehlen, UI-Dateien, Logs, Abhängigkeiten oder Übersetzungen gemacht werden.

## Zweck dieses Repositories

Dieses Companion-Repo dient dazu, sichere Arbeitsmaterialien vorzubereiten:

- deutschsprachige Einordnung von MiroFish,
- Jan-spezifische Workflows,
- anonymisierte Seed-Daten,
- Prüfvorlagen für MiroFish-Reports,
- Codex-Aufgabenlisten,
- Datenschutz- und Sicherheitsregeln.

## Was MiroFish hier leisten soll

MiroFish wird in diesem Repo als Werkzeug für qualitative Analyse gerahmt:

- Szenarioanalyse,
- Strategieanalyse,
- Mustererkennung,
- Beziehungsknoten und Kontextverbindungen,
- Hypothesenbildung aus kuratierten Dokumenten.

MiroFish-Ergebnisse sind in diesem Arbeitsraum immer als **Hypothesen, Szenarien oder Interpretationen** zu behandeln. Sie ersetzen keine Faktenprüfung, keine Rechtsberatung, keine Finanzberatung und keine fachliche Validierung.

## Was MiroFish hier nicht ist

MiroFish wird in diesem Repo ausdrücklich **nicht** als Werkzeug für folgende Zwecke dargestellt:

- Aktienprognosen,
- exakte Zahlenprognosen,
- Wahlprognosen,
- medizinische oder rechtliche Entscheidungen,
- präzise Vorhersagen komplexer Ereignisse,
- produktive Entscheidungen ohne menschliche Prüfung.

## Sicherheitsregeln

- Keine API-Keys committen.
- Keine echten privaten Chatlogs verwenden.
- Keine sensiblen Rohdaten einfügen.
- Daten vor der Nutzung anonymisieren oder synthetisch erzeugen.
- Externe API-, LLM- oder Memory-Dienste klar markieren.
- Alle MiroFish-Ergebnisse als Hypothesen oder Szenarien kennzeichnen.

## Aktueller Status

Dieses Repository ist ein vorbereitender Companion-Arbeitsraum. Installationsvoraussetzungen und Startbefehle für MiroFish werden hier erst dokumentiert, wenn ein echtes Upstream-Repository, ein geprüfter Fork oder ein freigegebener Submodule-Import vorliegt.
